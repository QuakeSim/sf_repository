Welcome to the QuakeSim2 Portal Capabilities include
  • Facilities for accessing real-time and archival GPS data
  • Time series analysis tools (RDAHMM).
  • Mesh generation and finite element simulation tools (GeoFEST).
  • Real time GPS position values.
  • Okada-based methods for calculating displcements and modeling faults (Disloc and Simplex).
  • Access to the QuakeTables fault database.
For status updates and other information, click the News tab above.

For more information, see www.quakesim.org and www.crisisgrid.org.
