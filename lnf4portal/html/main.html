Welcome to the QuakeSim2 Portal

This portal revises the original QuakeSim portal so that it complies with Java portlet standards. Current portlets include

  • Facilities for accessing real-time and archival GPS data
  • Time series analysis tools, including ST_Filter and RDAHMM.
  • Mesh generation and finite element simulation tools (GeoFEST).
  • Real time GPS position values.
  • Okada-based methods (Disloc and Simplex).

For more information, see www.quakesim.org and www.crisisgrid.org.

If you don't have an account, you can log in as guest/guest.
